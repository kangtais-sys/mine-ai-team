(주)밀리어스(이하 "회사")는 개인정보보호법에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 하기 위하여 다음과 같이 개인정보처리방침을 수립하여 공개합니다.

1. 개인정보의 수집 항목 및 수집 방법

수집 항목수집 방법
쇼핑몰 주문 데이터 (주문번호, 주문일시, 주문금액, 상품정보)카페24 API 연동
매출 현황 데이터 (일별/월별 매출 집계)카페24 API 연동

※ 고객 개인정보(이름, 연락처, 주소 등)는 수집하지 않습니다.

2. 개인정보의 수집 및 이용 목적

  • 자사 내부 매출 현황 분석 및 대시보드 표시
  • 상품 판매 추이 모니터링
  • 경영 의사결정 지원

3. 개인정보의 보유 및 이용 기간

수집된 데이터는 서비스 이용 기간 동안 보유하며, 서비스 해지 시 즉시 파기합니다.

4. 개인정보의 제3자 제공

회사는 수집한 데이터를 제3자에게 제공하지 않습니다. 내부 직원 전용 대시보드에서만 사용됩니다.

5. 개인정보의 파기 절차 및 방법

보유 기간이 경과하거나 처리 목적이 달성된 경우, 해당 데이터를 지체 없이 파기합니다. 전자적 파일은 복구 불가능한 방법으로 삭제합니다.
